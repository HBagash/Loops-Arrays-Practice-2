While Loop Cheat Sheet

A loop runs while a condition is True.

# 1. Setup variable
count = 0

while count < 5: # 2. Condition & Colon
    print("Looping!") # 3. Indented Action

    count += 1 # 4. Update (Avoid Infinite Loop!)

The Must-Haves

  • ✅ ":" After the condition
  • ✅ Tab key to indent code
  • ✅ Update the counter

Avoid These

  • ❌ Forgetting the colon
  • ❌ Un-indented code
  • ❌ Infinite loops

Arrays Cheat Sheet

Arrays (Lists) store multiple items in one variable.

# Create a list
names = ["A", "B", "C"]

# Access index 0 (which is "A")
print(names[0])

# Iterating with a while loop
i = 0
while i < len(names):
    print(names[i])
    i += 1

The Key Rules

  • 📍 Indexing starts at 0
  • 📦 Use [ ] for indexes
  • 📏 Use len() for size

Useful Methods

  • ➕ .append(item)
  • 🔄 list[i] = new
  • 🔢 range(len(list))
